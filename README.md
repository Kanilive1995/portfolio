# React + Vite Portfolio Template

A minimal React setup powered by **Vite**, designed for fast development with Hot Module Replacement (HMR) and clean, maintainable code.

## Key Features

- Fast React development with Vite HMR
- ESLint configured for code quality and best practices
- Compatible with official React plugins:
  - `@vitejs/plugin-react` – Babel-based Fast Refresh
  - `@vitejs/plugin-react-swc` – SWC-based Fast Refresh

> Note: The React Compiler is optional due to potential performance impact.

## Highlights

- Optimized for modern frontend development
- Easily extendable with TypeScript and advanced ESLint rules
- Ready to integrate CSS frameworks or component libraries
